# Piper
Pi based audio player geared towards providing uninterrupted audio ambience. Loops multiple audio files concurrently, and exposes a webapp for live control over track volume, panning, and other tooling.

## Setup
- update system (optional)
  ```sh
  sudo apt update
  sudo apt upgrade
  sudo apt autoremove
  ```
- Run install
  ```sh
  sudo ./install.sh
  ```
